# Spore-Inspired Space Stage

Трехмерная сцена на Three.js, вдохновленная космическим этапом Spore. Проект собирается и запускается через Vite.

## Установка

```bash
npm install
```

## Режим разработки

```bash
npm run dev
```

Dev-сервер Vite поднимет локальный хост (по умолчанию `http://localhost:5173`). Откройте страницу в браузере, чтобы попасть на галактическую карту.

## Управление

- **Карта галактики** — кликайте по звездам, чтобы перемещать корабль; прокрутка вниз переводит в выбранную систему, прокрутка вверх возвращает обратно.
- **Солнечная система** — кликайте по планетам, чтобы переправиться к ним; прокрутка вниз при нахождении возле планеты переносит на орбиту, прокрутка вверх возвращает к карте системы.
- **Орбита планеты** — используйте `W`, `A`, `S`, `D` для перемещения по сфере, колесо мыши регулирует высоту орбиты.

## Сборка production-версии

```bash
npm run build
```

Собранные файлы появятся в каталоге `dist/`. Для предпросмотра можно запустить:

```bash
npm run preview
```
